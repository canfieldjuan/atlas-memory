# Atlas Video Processing System

This project is a distributed system for real-time video processing from various sources like drones, RC cars, and cameras. It's designed to be a modular and extensible platform for drone management, object tracking, and intelligent data processing.

## Components

- **Ingest:** Clients for capturing video from sources.
- **Processing:** Real-time video processing and object detection.
- **Control:** Drone fleet management and mission planning.
- **Persistence:** Data and storage for video and metadata.
- **LLM Service:** For high-level data fusion and intelligent routing.
